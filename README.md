# github-training-project

## 🛠️ Tech Stack

### Programming Languages:
- HTML5
- CSS3
- JavaScript (ES6+)

### Tools & Libraries:
- **Font Awesome**: For icons.
- **Google Fonts**: (Implicitly used, not explicitly listed but typical for modern web pages).

## 🎨 Design System
### Colors:
- **Primary Accent**: `#1689E7` (Used for buttons, icons, and active states)
- **Background**: `#F6F8FC` (Main page background)
- **Card Background**: `#FFFFFF` (White)
- **Text Colors**: Black (`#000`) and Grey (`#666`, `#999`)

### Typography:
- **Primary Font**: `Barlow Semi Condensed`
- **Fallback Fonts**: `Arial`, `sans-serif`
- **Font Sizes**: Varied, with `45px` for main headings and smaller sizes for text.

### Spacing & Layout:
- Uses a **Flexbox** and **Grid** system.
- **Sidebars** (`aside` and `main`) are fixed or sticky.
- **Content Area** uses a responsive grid for project cards.

## 🧩 Core Features
1.  **Sidebar Navigation**: Collapsible navigation with icons.
2.  **Project Management**: Dashboard for viewing and managing projects.
3.  **Profile Section**: User profile with image, name, and location.
4.  **Search Functionality**: Global search bar in the header.
5.  **Notifications**: Bell icon with a dropdown list for notifications.
6.  **Action Buttons**: Share, Add Task, Filter, and Sorting options.
